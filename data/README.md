# Data

Store here:

- **Numbers & sources** (user counts, downloads, GMV, search volume, etc.)
- **Charts / visuals** used in deliverables or presentation
- **Raw research data** (exported tables, links to spreadsheets)

Keep filenames clear (e.g. `competitor-metrics.csv`, `skill-demand-sources.md`) and cite these files in the main deliverable `.md` files where relevant.
