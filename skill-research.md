# Deliverable 2: Skill Research (25%)

List **5–10 skills** the Skill Market **should** have. Each skill must have evidence that users **need** it and it will **create demand** (attract users, drive share purchases for private access).

## Per-skill requirements

| Field | Must answer |
|-------|-------------|
| **Skill name** | Short, clear (e.g. "Whale Wallet Tracker", "Auto Yield Optimizer") |
| **Target user** | Who uses it? (e.g. "Retail DeFi trader, portfolio $5K–$50K") |
| **Problem** | What pain? (e.g. "Spend 3h/day manually tracking whale wallets on Etherscan") |
| **Current alternative** | What do they use today? (e.g. "Nansen $100/mo, Arkham free but limited") |
| **How the skill solves it** | Short description of how the skill works |
| **Visibility & monetization** | Public or Private (holder-gated)? If private, min shares? |
| **Demand evidence** | Search volume, forum posts, Twitter, existing tools, on-chain data (with sources) |
